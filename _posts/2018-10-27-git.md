# Git 常用命令



## 一. tag

1. 查看tag: `git tag`
2. 创建tag: `git tag 0.1`(`git tag -a 0.1 -m 'add v0.1'`)
3. 查看tag变更内容: `git show 0.1`
4. 删除tag: `git tag -d 0.1`
5. 推送到远程：`git push origin 0.1 `
6. 删除远程tag: `git push origin :refs/tags/0.1  `
