## 常用shell脚本

### 遍历目录并查找文件到某个目录

```bash
src=/usr/local/201707/*
dest=/data/home/bensonlin/201707

for file in $src
do
if [ -f "$file" ]
then
  a=`cat $file | grep '<'IP'>'`
  if [ "$a" ]
  then
   cp -v $file $dest
  fi
elif [ -d "$file" ]
then
  echo "$file is directory"  
fi
done
~
~
```
